### Enable GitHub Pages (5 minutes)

The website is ready to deploy but needs GitHub Pages to be enabled:

1. **Go to Repository Settings**
   - Navigate to: https://github.com/charlieatkinson33/SimWiki/settings/pages

2. **Configure Pages**
   - Under "Build and deployment"
   - Set **Source** to: `GitHub Actions`
   - Click Save

3. **Wait for Deployment**
   - GitHub Actions will automatically build and deploy
   - Check the Actions tab to monitor progress
   - First build takes 2-3 minutes

4. **Access Your Site**
   - Once deployed, visit: https://charlieatkinson33.github.io/SimWiki/
   - Share this URL with the simulation community!
