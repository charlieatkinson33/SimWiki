# Assessment Tools

Effective assessment is crucial for simulation-based education. Commercial assessment tools can be expensive, but many alternatives exist for tracking learner performance and competency.

## Digital Checklists and Forms

### Commercial Options
- **SimCapture Assessment Module**: £2,000-£5,000/year
- **Custom Assessment Software**: £3,000-£8,000
- Features: Real-time scoring, analytics, reporting

### Free/Low-Cost Alternatives

#### Google Forms (Free)

**Features:**
- Unlimited forms
- Real-time responses
- Automatic spreadsheet export
- Shareable links
- Works on any device

**Use Cases:**
- Observation checklists
- Performance ratings
- Peer assessment
- Self-reflection forms
- Pre/post surveys

**Setup:**
1. Create form with checklist items
2. Share link with assessors
3. View responses in real-time
4. Export to Excel/Sheets for analysis

**Example Checklist Structure:**
- Header: Scenario name, learner ID, date
- Sections: Critical actions, communication, clinical reasoning
- Rating scales: Yes/No, Likert scales, open text
- Automatic scoring where applicable

#### Microsoft Forms (Free with Office 365)

**Advantages:**
- Integration with Microsoft tools
- Excel export
- Branching logic
- Quiz mode with auto-grading
- Teams integration

**Setup Similar to Google Forms**

#### REDCap (Free for Education/Research)

**Features:**
- Secure data collection
- Complex surveys
- Longitudinal tracking
- Audit trails
- Multi-site capability
- GDPR compliant

**Availability:**
- Many UK universities have institutional licenses
- Free for educational research
- Training available

**Best For:**
- Research projects
- Complex assessments
- Longitudinal tracking
- Multi-center studies

## Observer Tablets and Devices

### Commercial Options
- **iPad-based assessment systems**: £500-£1,000 per device
- **Dedicated assessment tablets**: £800-£1,500

### Budget Alternatives

#### Refurbished Tablets (£100-£300)

**Options:**
1. **iPad (older generation)** - £200-£300
   - Reliable and supported
   - Good battery life
   - Familiar interface

2. **Amazon Fire Tablets** - £50-£100
   - Very affordable
   - Basic but functional
   - Good for forms/checklists

3. **Android tablets** - £100-£200
   - Samsung Galaxy Tab A
   - Lenovo tablets
   - Various price points

**Setup:**
- Install assessment apps (Google Sheets, Forms apps)
- Configure WiFi
- Add bookmarks to assessment forms
- Protect with cases

**UK Suppliers:**
- Amazon Renewed
- MusicMagpie
- Back Market
- CEX (trade-in stores)

#### Smartphones as Assessment Tools

**Advantages:**
- Most people have them
- Always connected
- Camera for evidence
- Voice notes capability

**Use:**
- Personal devices (BYOD)
- Dedicated simulation phones (£50-£100)
- Old phones repurposed

## Timing Systems

### Commercial Options
- **Specialized simulation timers**: £200-£500
- Features: Large display, scenario phases, alerts

### DIY Alternatives

#### Visible Timer (Free-£30)

**Options:**
1. **Online timers** (Free)
   - Timer-tab.com
   - Online-stopwatch.com
   - Full screen on display
   - Visible to facilitators and learners

2. **Smartphone apps** (Free)
   - Timer+
   - Stopwatch apps
   - Cast to large display
   - Multiple simultaneous timers

3. **Physical timers** (£10-£30)
   - Large digital kitchen timers
   - Count-up or count-down
   - Loud alarms
   - Battery operated

#### Scenario Phase Timer (£50-£100)

**Components:**
- Tablet or small monitor - £50
- Custom timer webpage - Free
- Wireless control - Free

**Features:**
- Multiple phases (e.g., initial assessment, intervention, handover)
- Color-coded periods
- Alerts/warnings
- Visible to team

**Create with:**
- Simple HTML/JavaScript
- Google Slides with timer
- PowerPoint presentation

## Video Analysis Tools

### Commercial Options
- **Valt Analysis Module**: Included in system (£8,000+/year)
- **Coaching/analysis software**: £500-£2,000

### Free/Low-Cost Alternatives

#### Annotation Software

**Free Options:**

1. **ScreenPal (formerly Screencast-O-Matic)** - Free basic/£20 year
   - Record screen
   - Annotate videos
   - Add comments
   - Share online

2. **Epic Pen** (£30)
   - Draw on screen during playback
   - Highlight key moments
   - Simple interface
   - Windows/Mac

3. **VLC Media Player** (Free)
   - Slow motion playback
   - Frame-by-frame
   - Screenshot capability
   - Bookmarks

#### Video Tagging and Clips

**DaVinci Resolve** (Free)
- Professional video editor
- Cut specific moments
- Add annotations
- Create highlight reels
- Color coding
- Export various formats

**Use Cases:**
- Create highlight reels of key moments
- Compile exemplar performances
- Compare different attempts
- Build teaching libraries

#### Time-Stamped Notes

**Simple Google Doc Approach:**
1. Share document with observers
2. Note timestamp: "03:45 - Good communication"
3. Review synchronized with video
4. Collaborative notes from multiple observers

**Cost: Free**

## Competency Tracking

### Commercial Options
- **Learning management systems**: £5,000-£20,000/year
- **Competency tracking software**: £2,000-£10,000

### Free/Low-Cost Alternatives

#### Spreadsheet-Based (Free)

**Google Sheets or Excel:**
- Learner names as rows
- Competencies as columns
- Date achieved
- Score/rating
- Comments
- Visual dashboard with conditional formatting

**Features:**
- Multi-user editing (Google Sheets)
- Automatic calculations
- Charts and graphs
- Filter and sort
- Export options

**Template Example:**
```
Learner | Basic Life Support | Advanced Airway | IV Access | Date Completed
--------|-------------------|-----------------|-----------|----------------
Smith   | Pass             | Pass            | Not Yet   | 15/10/2025
Jones   | Pass             | Not Yet         | Pass      | 20/10/2025
```

#### Notion (Free for Education)

**Features:**
- Database functionality
- Kanban boards
- Calendar views
- Team collaboration
- Mobile apps

**Use Cases:**
- Track learner progress
- Schedule assessments
- Store documentation
- Link to resources

#### Trello (Free)

**Use for:**
- Visual progress tracking
- Competency cards
- Team assignments
- Checklist management

**Setup:**
- Board per cohort or program
- Lists for competency levels
- Cards for individual learners
- Move cards as they progress

**Use Cases:**

- Reflective practice
- Evidence collection
- Feedback documentation

## Objective Structured Clinical Examinations (OSCEs)

### Station Management

#### Paper-Based (£20-£50 per OSCE)

**Materials:**
- Printed checklists
- Clipboards - £3 each
- Timers - £10 each
- Signage - £10

**Process:**
- Examiners mark manually
- Transfer to spreadsheet
- Calculate scores
- Feedback to students

#### Digital OSCE (£50-£200 setup)

**Using Tablets:**
- Google Forms per station
- Automatic score calculation
- Real-time monitoring
- Instant results

**Setup:**
- One form per station
- Tablets for examiners
- Central monitoring
- Export for records

### OSCE Bells/Timers

**Commercial:** £200-£500

**DIY Options:**

1. **Smartphone App** (Free)
   - OSCE timer apps available
   - Connect to speakers
   - Customizable timings
   - Station announcements

2. **Computer System** (£50)
   - Timer software
   - Audio alerts
   - Visible countdown
   - Automated

## Data Security and Privacy

### GDPR Compliance

**Essential Practices:**
- Anonymize data where possible
- Secure storage (encrypted)
- Access controls
- Data retention policies
- Clear purposes

**Tools That Help:**
- Google Workspace (GDPR compliant)
- Microsoft 365 (GDPR compliant)
- REDCap (built for healthcare research)

### Secure Sharing

**Methods:**
- Password-protected links
- Specific user access only
- Time-limited shares
- Encrypted email

## Quality Assurance

### Inter-Rater Reliability

**Track Agreement:**
- Multiple assessors same scenario
- Compare scores
- Calculate kappa statistics
- Identify training needs

**Tools:**
- Excel with formulas
- Statistical software (free: R, JASP)
- Online calculators

### Calibration Sessions

**Process:**
- Watch recorded scenario together
- All rate independently
- Compare and discuss
- Improve consistency

**Cost: Time only**

## Tips for Success

### Form Design
- Clear instructions
- Logical flow
- Mix of question types
- Not too long
- Mobile-friendly
- Test before use

### Training Assessors
- Calibration sessions
- Clear criteria
- Practice scenarios
- Ongoing feedback
- Inter-rater reliability checks

### Data Management
- Regular backups
- Clear naming conventions
- Organized folders
- Version control
- Documentation
