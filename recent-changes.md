## ğŸ“ Understanding the Change Log

### Commit Messages

Each change (commit) has:
- **Title** - Brief description of the change
- **Author** - Who made the change
- **Timestamp** - When it was made
- **SHA** - Unique identifier
- **Files changed** - What was modified

### Viewing Changes

Click any commit to see:
- **Diff view** - Line-by-line comparison
- **Before/After** - What changed
- **Comments** - Discussion about the change
- **File tree** - All affected files

### Reverting Changes

If a change needs to be undone:
- Maintainers can revert any commit
- Creates a new commit that undoes the change
- All history is preserved
- Original change remains visible

## ğŸ” Search Changes

### Search Commits

Use GitHub's search to find specific changes:

- [Search all commits](https://github.com/charlieatkinson33/SimWiki/commits/main?q=search+term)
- Search by keyword, file, or author
- Filter by date range
- Use advanced operators

### Example Searches

- `author:username` - All commits by a user
- `path:categories` - Changes in categories folder
- `committer-date:2024-01-01..2024-12-31` - Changes in 2024
- Combine filters for precise results
